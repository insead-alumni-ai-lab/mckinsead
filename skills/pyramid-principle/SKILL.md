# Pyramid Principle (Minto)

## When to use
Stage 5 (Synthesis) and Stage 6 (Communication). Structure findings top-down.

## Core Rules
1. **Answer first**: Lead with the governing thought (the recommendation)
2. **Group and summarize**: Support with 3-5 key lines (MECE)
3. **Logically ordered**: Each group uses one of:
   - Deductive (major premise → minor premise → conclusion)
   - Inductive (similar items grouped by common theme)
4. **SCQA at the top**: Situation → Complication → Question → Answer

## Structure Template
```
Governing Thought (the answer)
├── Key Line 1 (MECE with 2 & 3)
│   ├── Evidence/Analysis A
│   └── Evidence/Analysis B
├── Key Line 2
│   ├── Evidence/Analysis C
│   └── Evidence/Analysis D
└── Key Line 3
    ├── Evidence/Analysis E
    └── Evidence/Analysis F
```

## Anti-Patterns
- ❌ Bottom-up (data → insight → recommendation) — always go top-down
- ❌ Topic labels instead of action titles
- ❌ More than 5 key lines (cognitive overload)
- ❌ Key lines that aren't MECE
