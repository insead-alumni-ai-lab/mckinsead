# Slide Grammar

## When to use
Stage 6 (Communication) — formatting slides per consulting standards.

## Slide Structure
1. **Action title**: Full sentence, ≤14 words, lead with the so-what
2. **Body**: ONE of:
   - Chart/visualization
   - Data table
   - 3-5 bullet evidence points
   - KPI dashboard
   - Key quote
3. **Footer**: Source citations

## Rules
- One message per slide (never cram two ideas)
- Never combine a chart AND a long bullet list
- Action titles are the takeaway — a reader skimming titles should get the full story
- Left-align everything (no centered text in body)
- Clean, minimal design — data speaks
