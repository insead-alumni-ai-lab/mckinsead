# Chart Choice

## When to use
Selecting the right visualization for data in slides.

## Decision Tree
| Data Shape | Chart Type |
|---|---|
| Parts of a whole | Waterfall, Stacked bar, Pie (≤5 items only) |
| Comparison across categories | Bar chart (horizontal for many items) |
| Change over time | Line chart, Area chart |
| Distribution | Histogram, Box plot |
| Correlation | Scatter plot |
| Two-variable matrix | Heatmap, Marimekko |
| Flow/process | Sankey, Swimlane |
| Geographic | Choropleth map |
| KPI/metric | Big number card |

## Consulting Favorites
- **Waterfall**: Show bridges (e.g., revenue walk)
- **Marimekko**: Market share by segment
- **Tornado**: Sensitivity analysis
- **Bubble**: BCG-style portfolio positioning
