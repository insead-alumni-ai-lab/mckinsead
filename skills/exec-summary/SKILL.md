# Executive Summary

## When to use
First slide of any deck. Summarizes the entire recommendation.

## Structure
1. **SCQA recap** (2-3 sentences)
2. **Governing thought** (1 sentence — the recommendation)
3. **3-5 key supporting points** (one line each)
4. **Call to action** (what should happen next)

## Length
- 1 slide / 1 page maximum
- If it doesn't fit on one page, it's not an exec summary
